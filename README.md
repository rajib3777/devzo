# devzo Landing (React + Vite)

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

### Notes
- Color palette and typography match the provided Figma export (dark bg `#060606`, text `#F7F6F9`, accents `#CF2576` + `#8E23E2`).
- Framer Motion powers section reveals, headline word animation, and the custom cursor.
- Cards include cursor-based 3D tilt interaction.
